InSpace

3D Endless Runner

In "InSpace," players take on the role of a spacefaring hero trying to outrun a relentless alien force. Set in a vibrant and colorful universe, players must navigate through space avoiding the asteroids along the way. In addition to dodging obstacles, players can also pick up coins, increasing their high score.

Azurie

2D Endless Scroller

Azurie puts a fresh spin on Flappy Bird, featuring an azure bird soaring through serene obstacles. Built in Unity WebGL for browser-based play, this project showcases clean visual design and intuitive mechanics while incorporating cryptocurrency mining functionality. The game demonstrates expertise in both Unity development and blockchain integration, offering an engaging experience that seamlessly blends casual gameplay with crypto mining capabilities.

Discovering Africa

2D Platformer

Discovering Africa is an engaging 2D platformer developed during the 2022 Maliyo Games Game Jam, celebrating the rich cultural heritage of the African continent. Built in Unity for Android devices, players embark on an educational adventure, navigating through beautifully illustrated levels inspired by diverse African landscapes and landmarks. The game combines precise platforming mechanics with cultural storytelling elements, challenging players to collect artifacts and learn about African history while avoiding obstacles. This passion project showcases mobile game optimization, responsive touch controls, and thoughtful integration of educational content within an entertaining gameplay framework.

Platform Rush

Platform Rush

Platform Rush was my first project exploring 2D game mechanics in Unity, built for PC and inspired by the classic game Bounce. In this fast-paced platformer, players navigate through a series of dynamic levels filled with obstacles, using precise timing and movement to overcome challenges. This project introduced me to foundational game development concepts, including physics-based mechanics, level design, and responsive controls, marking the start of my journey as a game developer.
